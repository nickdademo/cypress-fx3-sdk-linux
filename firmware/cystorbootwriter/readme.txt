Firmware Programmer utility Firmware for Storage using SCSI APIs
-------------------------------------------------------------------

This firmware application is used by the CyStorProg utility as the programming 
firmware to partition and download firmware images to storage (MMC/eMMC).

The utility only works with SPort0.
